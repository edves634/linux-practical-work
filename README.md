# Практическое задание по Linux

## Тема урока: Работа с пользователями и группами в Linux

### Задание:

1. **Создать пользователя в системе.** Добавить его в группу wheel и задать пароль. Проверка: команда `su user` выводит нас в консоль под новым пользователем.

2. **Используя ssh-keygen под этим пользователем сгенерировать пару SSH ключей.** В домашней директории пользователя, в папке `~/.ssh` должны быть два файла с ключами.

3. **Используя пакетный менеджер установить редактор nano.** Если он уже есть в системе, его можно сначала удалить, потом поставить. Результат: команда `nano` должна работать.

4. **Отредактировать файл `/etc/hosts`** добавить `127.0.0.1 Yandex.ru`, проверить `ping Yandex.ru` и убедиться, что подмена сработала.
